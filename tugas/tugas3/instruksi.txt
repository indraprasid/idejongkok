Program aplikasi karyawan sederhana menggunakan konsep Object Oriented Programming dengan ketentuan sebagai berikut: 

- Setiap karyawan, memiliki attribute name, position, dan salary. 
- Untuk salary adalah confidential.
- Setiap karyawan dapat ditampilkan informasi karyawan tersebut. 
- Setiap karyawan memiliki method untuk menaikkan gaji dengan persentase sebagai berikut: 
# Position IT = 10% 
# Position HR = 15% 
# Position Director = 1%

Catatan:
- Program aplikasi dapat digunakan dengan baik oleh pengguna lewat terminal / python console. 
- Program aplikasi terdiri dari 2 file, yaitu 1 file module dan 1 file execution.
- Program aplikasi dapat dijalankan dengan .\main.py
- Berikut adalah contoh inputannya:
Masukkan Nama: Indra
Masukkan posisi (IT/HR/Director): it
Masukkan Gaji: 15000000

- Berikut adalah contoh outputnya:
=== Informasi Karyawan ===
Nama: Indra, Posisi: IT
Gaji Indra adalah Rp 15,000,000.00

=== Proses Kenaikan Gaji ===
Gaji Indra telah diperbarui menjadi Rp 16,500,000.00
